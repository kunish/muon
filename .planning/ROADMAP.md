# Muon - Project Roadmap

## Project Overview
Muon is a Matrix-protocol based desktop chat application built with Vue 3 + Tauri, featuring a Lark/Discord-inspired UI with comprehensive chat, calls, and contacts functionality.

## Vision
Evolve Muon into a full-featured collaboration platform with Discord-style server/channel hierarchy, enhanced social features, and enterprise capabilities.

## Current State
- Core chat module: 50+ components, fully functional
- Audio/video calls: LiveKit-based, working
- Contacts management: functional
- Settings: comprehensive
- Matrix protocol layer: 14 modules, well-abstracted

## Planned Enhancements
- Discord-inspired social features (custom status, friend system, blocking)
- Server/Space concept (Matrix Spaces)
- Channel hierarchy and permissions
- Message Markdown rendering
- Slowmode for channels
